Windows:
.\make.bat setup // create env + install requirements
venv\Scripts\activate // enter env
.\make.bat clean // delete env
py main.py // launch app


Linux(not suported yet):
make setup
make run
make clean

General Commands:
pip install -r requirements.txt // install all packages within the requirements.txt file
pip install <package> // for installing new packages (look below)
pip freeze > requirements.txt // after installing new packages
